برنامج يحتوى على عشرة أذكار
ثابتة عن النبي ﷺ
من الكتاب والسنة
يساعد على المداومة عليها 
و بالتالى حفظها بسهولة
---------------------
This program contains ten authentic supplications (adhkar) from the Prophet ﷺ, 
taken from the Qur’an and the Sunnah, 
which helps users consistently practice them and easily memorize them.
